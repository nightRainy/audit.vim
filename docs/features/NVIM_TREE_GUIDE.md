# nvim-tree 文件树使用指南

## 安装完成 ✓

nvim-tree.lua 已成功安装并配置！

## 快捷键

### 打开/关闭文件树
- **`<leader>e`** (空格 + e): 切换文件树显示/隐藏
- **`<leader>E`** (空格 + Shift + e): 在文件树中定位当前文件

## 文件树操作

### 导航
- `Enter` 或 `o`: 打开文件或展开/折叠目录
- `<C-v>`: 在垂直分割窗口中打开文件
- `<C-x>`: 在水平分割窗口中打开文件
- `<C-t>`: 在新标签页中打开文件
- `P`: 跳转到父目录
- `-`: 跳转到上一级目录

### 文件操作
- `a`: 创建新文件（以 `/` 结尾创建目录）
- `d`: 删除文件或目录
- `r`: 重命名文件
- `x`: 剪切文件
- `c`: 复制文件
- `p`: 粘贴文件
- `y`: 复制文件名
- `Y`: 复制相对路径
- `gy`: 复制绝对路径

### 视图控制
- `R`: 刷新文件树
- `H`: 显示/隐藏隐藏文件
- `I`: 显示/隐藏 gitignore 文件
- `W`: 折叠所有目录
- `E`: 展开所有目录
- `.`: 切换到项目根目录

### 搜索和过滤
- `f`: 过滤文件（输入过滤条件）
- `F`: 清除过滤
- `/`: 搜索文件名
- `q`: 关闭文件树

### Git 集成
- 文件旁边显示 Git 状态图标：
  - `✓`: 已暂存
  - `✗`: 未暂存修改
  - `★`: 未跟踪文件
  - ``: 未合并
  - `➜`: 重命名
  - ``: 删除

### 诊断集成
- 文件旁边显示 LSP 诊断图标：
  - ``: 错误
  - ``: 警告
  - ``: 信息
  - ``: 提示

## 使用示例

1. **打开文件树**: 按 `空格 + e`
2. **导航到文件**: 使用 `j/k` 或方向键
3. **打开文件**: 按 `Enter`
4. **创建新文件**: 按 `a`，输入文件名
5. **创建新目录**: 按 `a`，输入目录名并以 `/` 结尾
6. **查找当前文件**: 按 `空格 + Shift + e`

## 与 audit.vim 集成

nvim-tree 完美配合 audit.vim 使用：
- 使用 `python3 avim.py open` 打开项目
- 按 `空格 + e` 打开文件树浏览项目结构
- 使用 `F2` 搜索符号，`<leader>fs` 查找引用
- 使用 `<leader>o` 打开符号大纲（Aerial）
